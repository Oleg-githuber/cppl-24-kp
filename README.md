# Парсер ini-файлов
Чтение данных из файла производится построчно. Каждый раз при чтении строки производится инкремент счётчика строк. 
Данные хранятся в контейнере map с ключём - именем секции, и значением - контейнером map с ключём - именем переменной, и значением - 
значением переменной в виде строки.
Используется запрет на переопределение переменных.
Получение значения переменной в виде выбранного типа данных происходит с промежуточным преобрадованием строки в double.
